## [2026-06-14] TODO：設「自動恢復」——讓重開機也能每晚穩定觸發（此帳號無密碼，可全自動）

帳號 `user` **無密碼** → 自動登入可直接設、沒有密碼外洩風險。下面三件事都需系統管理員權限，
請用一個「以系統管理員執行」的 PowerShell **一次做完**（會跳 UAC，按「是」即可，無密碼不需輸入）：

```powershell
# 1) 自動登入（開機自動進桌面；空密碼）
$k="HKLM:\SOFTWARE\Microsoft\Windows NT\CurrentVersion\Winlogon"
Set-ItemProperty $k AutoAdminLogon 1
Set-ItemProperty $k DefaultUserName "user"
Set-ItemProperty $k DefaultPassword ""
# 2) 永不睡眠/休眠（接電源）
powercfg /change standby-timeout-ac 0
powercfg /change hibernate-timeout-ac 0
powercfg /change monitor-timeout-ac 0
powercfg /hibernate off
# 3) 排程補跑錯過的（錯過 21:00 → 下次可用時補跑）
$t=Get-ScheduledTask -TaskName QT-Nightly-5090
$t.Settings.StartWhenAvailable=$true
Set-ScheduledTask -TaskName QT-Nightly-5090 -Settings $t.Settings
```

**驗證**：
- `(Get-ItemProperty $k).AutoAdminLogon` = 1
- `powercfg /query` 看 AC 待機逾時為 0
- `(Get-ScheduledTask QT-Nightly-5090).Settings.StartWhenAvailable` = True

若自動登入登錄機碼法無效（有些 Windows 版本擋空密碼自動登入），改用備援：執行 `netplwiz` → 取消勾「使用者必須輸入使用者名稱和密碼」→ 套用（空密碼直接按確定）。
